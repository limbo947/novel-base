# 大纲设计方法卡

> 来源：《苟在武道世界成圣》 | 作者：在水中的纸老虎

---

## 这个模块最值得提炼什么

**"地图升级 + 境界跨越 + 势力博弈"的三层螺旋大纲结构**——如何让长篇网文在百万字级别保持叙事张力和升级感，避免"换地图=换书"的割裂感和"原地踏步"的重复感。

本书108万字的大纲呈现出清晰的三层螺旋：外层是地图/势力的横向扩展，中层是武道境界的纵向攀升，内层是主角人设的深层展开。三层不是独立推进，而是互相咬合、互相驱动。

---

## 具体拆解

### 1. 地图升级的节奏与衔接

全书4次地图切换，每次切换遵循"推拉节奏"：

```
旧地图积累 → 触发事件 → 被迫/主动离开 → 新地图适应期 → 新地图立足 → 新地图突破 → 循环
```

| 地图切换 | 触发事件 | 衔接方式 | 新旧地图关联 |
|----------|----------|----------|-------------|
| 哑子湾→五台派 | 暗杀宋铁后需要更高平台 | 考核选拔 | 哑子湾的武馆基础成为五台派的起步优势 |
| 五台派→天宝上宗 | 擂台赛一鸣惊人后被选中 | 宗门推荐 | 五台派的修炼成果直接转化为天宝上宗的竞争力 |
| 天宝内斗→古国遗址 | 探险任务 | 任务驱动 | 天宝上宗内斗的势力格局延续到探险中 |
| 万法峰主→宗主 | 宗主之位空缺 | 选举/争夺 | 之前积累的人脉和实力成为争夺资本 |

**关键技巧**：每次地图切换都不是"突然跳到新世界"，而是旧地图的积累在新地图中有直接用处。这避免了"换地图=重置"的割裂感。

### 2. 境界跨越的节奏设计

武道八阶的跨越不是均匀分布的：

| 境界 | 大致章节区间 | 跨越方式 | 篇幅占比 |
|------|-------------|----------|----------|
| 明劲 | 1-10章 | 苦练突破 | 约5% |
| 暗劲 | 10-30章 | 战斗中突破 | 约8% |
| 化劲 | 30-60章 | 宗门资源+苦练 | 约10% |
| 抱丹 | 60-100章 | 秘境历练 | 约13% |
| 罡劲 | 100-180章 | 多种真气合一 | 约24% |
| 真元 | 180-350章 | 十次淬炼 | 约50% |
| 宗师 | 350-450章 | 古国遗址生死战 | 约30% |
| 元神 | 450章+ | 未完成 | — |

**关键技巧**：越往高阶，每个境界跨越所需的篇幅越长、难度越大。低阶境界（明劲-化劲）快速跨越以保持节奏，高阶境界（真元-宗师）大幅拉长以制造期待感和仪式感。

### 3. 势力博弈的层层嵌套

势力格局不是简单的"打怪升级"，而是层层嵌套的博弈结构：

```
地方帮派（金河帮/老虎帮）
  └─ 宗门（五台派）
       └─ 上宗（天宝上宗）
            └─ 联盟（各大上宗之间）
                 └─ 天下格局（还源教/鬼巫宗等）
```

每一层势力都有：
- **内部派系**：五台派有不同山峰的竞争，天宝上宗有各峰主的博弈
- **外部对手**：同层级的其他势力
- **上位势力**：更高层级的势力作为"天花板"
- **下位势力**：已超越的旧势力作为"参照物"

### 4. 主线与支线的编织方式

| 支线类型 | 代表 | 与主线的关系 | 处理方式 |
|----------|------|-------------|----------|
| 情感支线 | 表姐赠银、发小离散 | 提供动机和温度 | 在关键时刻回收，不做长线拖拽 |
| 势力支线 | 宗门政治、世家站队 | 提供障碍和资源 | 与武力升级交替推进 |
| 悬疑支线 | 鬼巫宗人丹阴谋 | 提供中期驱动力 | 逐步揭露，最终引爆 |
| 成长支线 | 师徒关系、同门竞争 | 提供对比和张力 | 随地图切换自然更替 |

---

## 为什么有效

1. **三层螺旋避免单调**——读者不会觉得"只是在升级"或"只是在换地图"，因为三层同时在推进
2. **旧积累在新地图有效**——避免了"换地图=重置"的读者流失
3. **境界跨越节奏不均匀**——低阶快、高阶慢，符合"越往上越难"的直觉，也制造了期待感
4. **势力嵌套提供持续博弈**——不会因为打完一个boss就没事做了
5. **支线与主线深度绑定**——没有"废支线"，每条支线都服务于主线推进

## 可复用写法

### 写法1：三层螺旋大纲法

设计大纲时，同时规划三个层级：
- **外层**：地图/势力切换计划（每100-150章一次）
- **中层**：境界跨越计划（低阶快、高阶慢）
- **内层**：人设展开计划（每阶段揭示主角的一个新侧面）

三层必须在每个阶段都有交叉点，不能独立推进。

**检查清单**：
- [ ] 每次地图切换，旧地图的积累在新地图有直接用处？
- [ ] 每次境界跨越，都有外部压力（不是闭关就能突破）？
- [ ] 每个势力层级，内部都有派系博弈？

### 写法2：境界跨越的"不均匀节奏"法

低阶境界（前30%）用全书20%的篇幅快速跨越，中阶境界（30-70%）用50%的篇幅稳步推进，高阶境界（后30%）用30%的篇幅大幅拉长。每个高阶跨越都要有"生死战"级别的压力。

### 写法3：势力嵌套的"俄罗斯套娃"法

设计势力格局时，确保每一层势力都有内斗、外敌和上位压力。当主角解决当前层级的问题后，自然暴露出上一层的更大问题，形成"打完小怪发现boss"的持续推进感。

## 可量化参数

| 参数 | 建议值 | 本书实际值 |
|------|--------|-----------|
| 地图切换间隔 | 100-200章 | 约120-150章 |
| 每个大地图的章数 | 80-200章 | 50-350章（后期拉长） |
| 境界跨越间隔（低阶） | 10-30章 | 10-30章 |
| 境界跨越间隔（高阶） | 50-150章 | 50-200章 |
| 势力层级数 | 3-5层 | 4层 |
| 支线与主线的交叉频率 | 每10-20章 | 约15章 |
| 主线推进占比 | 60-70% | 约65% |

## 适用边界

**适用场景**：
- 100万字以上的长篇网文大纲设计
- 有升级体系的作品（武道/修仙/玄幻）
- 需要多地图切换的冒险类故事
- 有势力博弈元素的作品

**不适用场景**：
- 50万字以下的短篇/中篇（不需要这么复杂的结构）
- 纯日常/恋爱/悬疑类（不需要升级体系）
- 单地图深耕型作品（如校园文、职场文）
