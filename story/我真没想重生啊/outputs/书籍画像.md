# 书籍画像：《我真没想重生啊》

> 作者：柳岸花又明 | 篇幅：长篇（百万字级） | 题材：都市重生、年代文、校园

---

## 一、一句话定位

一部以2002年为起点的都市重生小说，主角陈汉升带着未来记忆回到高三毕业之际，在商战和感情之间展开的成长故事。

## 二、核心人设

| 维度 | 内容 |
|------|------|
| 主角 | 陈汉升，35岁重生回2002年的高三毕业生，成熟稳重且有商业头脑，外表玩世不恭内心细腻 |
| 金手指/核心设定 | 重生带来的未来记忆，对时代发展趋势的了解 |
| 行为模式 | 果断决策、善于利用信息差、重视亲情友情 |
| 道德光谱 | 偏向正义，有原则但灵活变通 |
| 成长弧线 | 从普通学生到商业精英的转变，情感上的成熟 |

## 三、世界观与体系

以2002年为背景的现实世界，涉及校园生活、商业发展、社会变迁等多个维度，展现了从2002年到现代的时代变迁。

## 四、叙事结构特征

采用对比式开篇，先展示未来成功状态，再回到起点；以年代发展为时间线，融合商战、感情、校园等多条线索。

## 五、核心爽感机制

重生带来的信息差优势，商业成功的成就感，感情线的发展，以及与时代同步成长的满足感。

## 六、情感锚点体系

亲情：与家人的关系
友情：与发小的互动
爱情：与沈幼楚、萧容鱼的感情纠葛
事业：商业成功的追求

## 七、权谋博弈特征

商业竞争中的策略运用，人际关系的处理，时代机遇的把握。

## 八、写作风格总结

| 维度 | 特征 |
|------|------|
| 文风 | 轻松幽默，略带玩世不恭，现实主义 |
| 节奏 | 快速，开篇即高潮，情节推进紧凑 |
| 对话 | 自然流畅，符合人物性格 |
| 战斗 | 较少，以商业竞争和人际关系为主 |
| 心理 | 细腻真实，展现主角内心活动 |
| 幽默 | 适度，增添作品趣味性 |

## 九、最适合参考什么

对比式开篇的运用，时空锚点的设置，人物快速定位的方法，年代感的营造，商业题材与感情线的平衡。

## 十、最不适合照搬什么

过度依赖重生金手指，忽略人物成长的合理性，年代细节的准确性。
