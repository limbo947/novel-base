# 提取卡 0128

## 叙事功能

- **核心冲突**：杨间与其他驭鬼者在培训基地遭遇鬼差的袭击，试图与钱毅汇合，同时引走鬼差
- **信息揭示**：人皮纸提供新的计划（取走鬼身上的棺材钉，削减鬼的数量）；鬼差的真实身份是被鬼占据的卫景尸体；总部决定集体行动前往实验室研究鬼棺
- **人物关系**：杨间与张雷、王江、黄子雅等驭鬼者的合作；杨间对钱毅的不满；总部人员的团结与决策
- **世界观构建**：通过鬼差事件，展示灵异世界的恐怖和复杂性，驭鬼者面对强大厉鬼的无力感，以及团队合作的重要性

## 关键内容

1. **杨间的行动**：
   - 杨间带着众人继续前进，寻找其他驭鬼者，引走鬼差
   - 杨间的鬼手开始侵蚀他的胳膊，失去知觉
   - 杨间决定与钱毅汇合，尽管钱毅之前欺骗了他

2. **总部的决策**：
   - 王小明提议去实验室研究鬼棺，认为鬼棺与鬼差可能存在联系
   - 童倩、冯全、李军等决定陪同王小明前往实验室
   - 副部长曹延华决定让所有驾驭两只鬼的驭鬼者集体行动，护送王小明去实验室

3. **人皮纸的计划**：
   - 人皮纸透露与钱毅汇合时的危险：鬼会袭击钱毅，平衡会被打破
   - 人皮纸提供新的计划：取走鬼身上的棺材钉，削减鬼的数量
   - 人皮纸暗示：鬼的身后就是活路

4. **钱毅的遭遇**：
   - 钱毅试图与杨间汇合，感到周围异常
   - 钱毅被鬼差袭击，脖子被鬼绳缠住，身体被撕开
   - 杨间带着众人赶到，准备救援

5. **鬼差的身份**：
   - 鬼差的形象是被鬼占据的卫景尸体
   - 卫景的尸体被鬼棺中的鬼占据，成为鬼的载体

## 方法候选

1. **灵异事件应对方法**：
   - **取走棺材钉**：取走鬼身上的棺材钉，削减鬼的数量，恢复平衡
   - **更改鬼的记忆**：使用染血的旧报纸更改鬼的记忆，让鬼误判自身数量
   - **分批次行动**：分出两批人，点燃鬼烛，吸引鬼的注意力

2. **危机应对策略**：
   - **团队合作**：驭鬼者聚集在一起，增加区域内鬼的数量，避免落单
   - **集体行动**：总部组织顶尖驭鬼者集体行动，研究鬼棺
   - **信息共享**：通过人皮纸获取关键信息和解决方案

3. **驭鬼者生存策略**：
   - **资源管理**：合理使用手中的灵异资源，如鬼烛、鬼手等
   - **风险分析**：分析方案的风险，做好应对准备
   - **相互支持**：在危机中相互帮助，共同应对危险

## 案例候选

1. **鬼差事件的应对**：
   - **触发条件**：鬼差（卫景厉鬼复苏）入侵培训基地，以落单驭鬼者为目标
   - **实施过程**：杨间点燃白色鬼烛吸引鬼差，与其他驭鬼者汇合，准备取走鬼身上的棺材钉
   - **结果**：暂时维持平衡，准备实施新的计划
   - **适用场景**：面对以落单者为目标的强大厉鬼，需要团队合作求生

2. **人皮纸的计划**：
   - **触发条件**：面临无解的灵异事件，需要寻找生存方法
   - **实施过程**：杨间询问人皮纸，获取新的计划
   - **结果**：人皮纸提供了取走棺材钉、削减鬼数量的计划
   - **适用场景**：面对无解的灵异事件，需要寻求超自然存在的帮助

3. **总部的集体行动**：
   - **触发条件**：S级灵异事件发生，威胁到大量驭鬼者的生命
   - **实施过程**：总部组织顶尖驭鬼者集体行动，护送王小明去实验室研究鬼棺
   - **结果**：正在实施中，尚未确定是否成功
   - **适用场景**：应对大规模灵异事件，需要集中资源和智慧

## 关键证据

1. **人皮纸的计划**：
   - "一旦和另外一位驭鬼者汇合，它或许会摆脱鬼烛的吸引，转而袭击那位驭鬼者，如果成功那么鬼的数量将逆转，平衡将会被打破，我们所有人将死在这里......"
   - "我必须借着那次的袭击主动出手，将鬼身上的某样东西取出来，如果能成功的话，我不但可以成功的削减那只鬼的数量，还有可能开启下一步的逃生计划，记住，只能削减鬼的数量，千万不要试图关押，否则厉鬼重启，一切将是徒劳的。"
   - "鬼的身后就是活路。"

   - "这次我打算让你们集体行动，在座但凡是驾驭了两只鬼的人全部护送王教授去实验室，不不但要取回鬼棺，还要取走实验室里的其他重要东西。"

4. **钱毅的遭遇**：

## 可学习元素

1. **生存智慧**：
   - 在危机中保持冷静，分析局势，寻找解决办法
   - 合理运用手中的资源和能力，应对不同类型的危机
   - 利用敌人的规律，制定针对性的应对策略

2. **危机应对**：
   - 建立有效的信息共享机制，及时传达危机信息
   - 在面对强大敌人时，采取主动策略，化被动为主动
   - 对危机的严重程度做出准确评估，避免盲目行动

3. **灵异世界规则**：
   - 了解鬼的能力和规律，寻找其弱点
   - 掌握不同灵异物品的使用方法，如鬼烛、鬼手等
   - 认识到灵异存在的复杂性，如鬼差的成长性和不可预测性

4. **人物互动**：
   - 在危机中关心同伴，相互支持
   - 与不同类型的人物打交道时，采取不同的应对策略
   - 建立团队信任，共同应对危机

5. **世界观构建**：
   - 通过历史事件和现有信息，推测灵异世界的真相
   - 认识到灵异存在的多样性和危险性
   - 理解驭鬼者在这个世界中的角色和责任
