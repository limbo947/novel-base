# 开篇规划方法卡

## 这个模块最值得提炼什么
- 这本书在开篇阶段最稳定的做法是什么：通过论坛故事引入核心灵异元素，逐步过渡到现实威胁，构建世界观
- 这本书最值得提炼的开篇抓手是什么：利用现代社交媒体场景增强真实感，让读者有代入感

## 开篇规划拆解
- 出场人物如何安排：主角杨间先出场，通过论坛互动展示其普通学生身份，然后引入周正等关键人物
- 主角第一次出场靠什么立住：普通高三学生的日常，通过论坛浏览展示其好奇心和正常反应
- 核心冲突在什么时候明确：第3-4章，黑衫老人出现在教室外，鬼域形成
- 第一章承担了哪些功能：引入主角，设置论坛故事钩子，铺垫灵异元素
- 前三章各自负责什么：第一章引入论坛故事，第二章引入周正的安全讲座，第三章冲突升级，鬼域形成

## 钩子与承诺
- 开篇钩子是什么：论坛里的鬼故事，包含照片和音频，增强真实感
- 它属于悬念型 / 危机型 / 反差型 / 利益承诺型哪种：悬念型 + 危机型
- 开篇先给了读者什么承诺：一个真实感强的灵异故事，后续会有更多恐怖事件
- 前三章第一次小兑现是什么：黑衫老人出现在教室外，论坛故事变为现实

## 为什么有效
- 这套开篇最强的抓手是什么：利用现代社交媒体场景，增强真实感和代入感
- 它为什么能让读者愿意继续看：逐步递进的恐怖氛围，从虚拟故事过渡到现实威胁，读者想知道后续发展
- 它优先锁住的是好奇心 / 爽感 / 危机感 / 代入感哪一种：代入感 + 好奇心

## 可复用写法
- 写法 1：利用日常场景（论坛、学校）引入核心元素，增强真实感
- 写法 2：通过照片、音频等多媒体元素增强故事的可信度
- 写法 3：逐步递进的恐怖氛围，从虚拟故事过渡到现实威胁

## 可量化参数
- 主角第几段出场：第2段
- 核心冲突第几章明确：第3-4章
- 开篇钩子出现位置：第1章前半部分
- 第一次小兑现在第几章：第3章
- 前三章平均净推进点数量：3-4个

## 适用边界
- 更适合什么题材 / 节奏：都市灵异、恐怖题材，快节奏
- 如果换成慢热文，哪些位置要后移：核心冲突明确的时间，鬼域形成的时间
- 哪些地方不能照搬：具体的论坛互动内容和灵异设定
